METHODOLOGY FOR BIOMASS HTT/HTL DATASET
========================================

This document provides comprehensive methodological details for researchers using the Biomass HTT/HTL Dataset in publications. Use the relevant sections when describing data sources, preprocessing, and quality assurance in your methodology.

========================================
1. DATA SOURCE AND COLLECTION
========================================

1.1 Literature Survey
---------------------
- Scope: Peer-reviewed publications on hydrothermal treatment/liquefaction of lignocellulosic and lignin-rich biomass
- Temporal coverage: 1982-2026 (40+ years)
- Total experimental runs: 3,693
- Source publications: 100+ peer-reviewed articles
- Process types covered: HTL, HTC, solvothermal liquefaction, related thermal conversions

1.2 Data Extraction Methods
----------------------------
- Tabular data: Direct extraction from published tables
- Graphical data: Digital extraction using plot digitization tools (WebPlotDigitizer)
- Supplementary materials: Retrieved from journal repositories
- Each data point tagged with provenance information (table, figure, digitized)

1.3 Data Standardization
-------------------------
- All yields normalized to dry feedstock basis unless otherwise specified
- Elemental compositions converted to dry or dry-ash-free basis
- Temperature units standardized to °C
- Time units standardized to minutes
- Pressure units standardized to MPa
- Energy values in MJ/kg

========================================
4. IMPUTATION METHODOLOGY
========================================

4.1 Machine Learning-Based Imputation
--------------------------------------
Method: Random Forest Regression
Applied to: Lignocellulosic composition (Lignin, Cellulose, Hemicellulose)
Training approach:
  - Features: C, H, O, Ash, Family_std, HHV
  - Train-test split: 80-20
  - Cross-validation: 5-fold
  - Performance: R² > 0.75 for Lignin, R² > 0.70 for Cellulose/Hemicellulose
  - Applied to ~4% of records missing structural composition

Imputation targets and coverage:
  - LCH_total: 100% coverage (95% measured, 5% imputed)
  - Lignin individual: 96.18% coverage
  - Cellulose/Hemicellulose: 95.61% coverage

4.2 Formula-Based Imputation
-----------------------------
Higher Heating Value (HHV):
  - Method: Channiwala-Parikh correlation
  - Formula: HHV = 0.3491*C + 1.1783*H + 0.1005*S - 0.1034*O - 0.0151*N - 0.0211*Ash
  - Units: MJ/kg
  - Applied when: Measured composition available but HHV not reported
  - Validation: Mean absolute error < 1.2 MJ/kg against reported values

Atomic Ratios:
  - O/C molar = (O_wt%/16) / (C_wt%/12)
  - H/C molar = (H_wt%/1) / (C_wt%/12)
  - Always calculated from elemental composition

Oxygen by Difference:
  - O_wt% = 100 - C - H - N - S - Ash
  - Applied when: Direct O measurement not reported
  - Flag: O_method = "by_difference"

4.3 Family-Specific Median Imputation
--------------------------------------
Applied to: Minor components, secondary properties
Method:
  - Group by Family_std
  - Calculate median value within family
  - Apply to missing values
  - Fallback to global median if family median unavailable

Used for:
  - Extractives content
  - Minor elemental components (N, S when family-consistent)
  - Secondary process conditions

4.4 Imputation Tracking
------------------------
All imputations documented via:
  - Boolean flags: *_imputed columns (e.g., Lignin_imputed)
  - Source indicators: *_source columns (e.g., LRI_imputed_source)
  - Method documentation: *_method columns
  - Values: "measured", "ML_RF", "formula", "median_family", "by_difference"

========================================
5. QUALITY ASSURANCE PROCEDURES
========================================

5.1 Elemental Mass Balance
---------------------------
Closure check: Sum of C + H + O + N + S + Ash
Thresholds:
  - Acceptable: 95-105 wt%
  - Warning: 90-95 or 105-110 wt%
  - Flagged: <90 or >110 wt%

Statistics:
  - Mean closure: 99.2%
  - Median closure: 99.8%
  - Within acceptable range: 94.3% of records
  - Standard deviation: 2.4%

5.2 Lignocellulosic Balance
----------------------------
Closure check: Sum of Lignin + Cellulose + Hemicellulose
Quality flags (LCH_closure_flag):
  - "OK": 85-105 wt%
  - "LCH>105": 105-110 wt%
  - "LCH>110": >110 wt%
  - "LCH<0": Negative values or sum <0

Distribution:
  - OK: 87.2%
  - Minor deviation (105-110): 9.1%
  - Major deviation (>110): 2.8%
  - Invalid (<0): 0.9%

Reasoning for tolerance:
  - Extractives not always reported separately
  - Measurement method variations
  - Proteins/lipids overlap with structural components

5.3 Yield Balance
-----------------
Total yield check: Bio-oil + Char + Gas + Aqueous
Expected: 85-105 wt% (losses to water, measurement uncertainty)
Statistics:
  - Mean total yield: 96.4%
  - Yield closure within 85-105%: 78.2% of cases with complete data
  - Common deficit: Gas phase often unreported or estimated

5.4 Energy Balance Validation
------------------------------
Energy recovery check:
  E_recovery = (Y_biooil * HHV_biooil + Y_char * HHV_char) / HHV_feed

Expected range: 50-95% (remainder to gas, aqueous, losses)
Statistics:
  - Mean energy recovery: 68.3%
  - Range: 45-88%
  - Validation: Cross-checked against reported values where available

5.5 Van Krevelen Diagram Validation
------------------------------------
Feedstock validation:
  - O/C ratio range: 0.4-0.9 (typical lignocellulosic)
  - H/C ratio range: 1.2-1.8
  - Outliers flagged for manual review

Bio-oil validation:
  - O/C ratio range: 0.1-0.6
  - H/C ratio range: 1.0-2.0
  - Expected trend: Lower O/C compared to feedstock

Char validation:
  - O/C ratio range: 0.05-0.5
  - H/C ratio range: 0.3-1.2
  - Expected: Progressive deoxygenation, dehydrogenation



Handling:
  - Physical impossibilities removed (e.g., yields >100%, negative values)
  - Extreme but plausible values retained with source verification
  - ~2.3% of data points flagged, ~0.8% removed after review

========================================
6. DERIVED DESCRIPTORS
========================================

6.1 Lignin Readiness Index (LRI)
---------------------------------
Formula: LRI = (Lignin/100 + C/60 + HHV/22 + (1/(O/C))/2 + (1/(H/C))/2) / 6

Purpose: Composite feedstock quality metric for HTL
Rationale:
  - Higher lignin content → better bio-oil yield
  - Higher carbon content → better energy density
  - Higher HHV → better energy recovery
  - Lower O/C → less oxygenated, more stable oil
  - Lower H/C → more aromatic character

Distribution:
  - Range: 0.65-1.15
  - Mean: 0.89
  - Kraft lignin: 0.95-1.10
  - Agricultural residues: 0.80-0.90
  - Algae: 0.70-0.80

6.2 Carbon Recovery Metrics
----------------------------
Carbon yield to bio-oil:
  C_yield_biooil (%) = (Y_biooil * C_biooil) / C_feed * 100

Carbon yield to char:
  C_yield_char (%) = (Y_char * C_char) / C_feed * 100

Total carbon recovery:
  C_total (%) = C_yield_biooil + C_yield_char

Statistics:
  - Mean C recovery to bio-oil: 48.3%
  - Mean C recovery to char: 24.1%
  - Mean total C recovery: 72.4%

6.3 Energy Metrics
------------------
Energy yield to bio-oil:
  E_yield_biooil (%) = (Y_biooil * HHV_biooil) / HHV_feed * 100

Energy densification factor:
  EDF = HHV_biooil / HHV_feed

Statistics:
  - Mean energy yield to bio-oil: 56.7%
  - Mean EDF: 1.42 (bio-oil is 42% more energy-dense)
  - Range EDF: 0.9-2.1

6.4 Compositional Indices
--------------------------
Holocellulose content:
  Holocellulose = Cellulose + Hemicellulose

Lignin share:
  L_share (%) = Lignin / (Lignin + Cellulose + Hemicellulose) * 100

Cellulose fraction:
  Ce_frac = Cellulose / (Cellulose + Hemicellulose)

========================================
8. DATA HARMONIZATION
========================================

8.1 Feedstock Naming Standardization
-------------------------------------
Method: Rule-based mapping + manual curation
Process:
  - Extract reported feedstock name
  - Map to standardized family (Family_std)
  - Maintain original name in Feedstock column
  - 180+ unique feedstocks → 15 major families

Example mappings:
  "Corn stover" → Agricultural Residue
  "Pine sawdust" → Softwood
  "Kraft lignin" → Kraft Lignin
  "Spirulina" → Algae

8.2 Catalyst Standardization
-----------------------------
Approach:
  - "none", "no catalyst", "blank" → standardized to "none"
  - cat_biomass_ratio set to 0.0 for non-catalytic runs
  - Catalyst names standardized (e.g., "Na2CO3", "sodium carbonate" → "Na2CO3")
  - Ambiguous cases flagged in tracking columns

8.3 Yield Basis Harmonization
------------------------------
Common reported bases:
  - Dry basis (db)
  - Dry ash-free (daf)
  - As received (ar)

Conversion strategy:
  - Target: All yields on dry basis
  - Moisture correction: Y_dry = Y_ar / (1 - Moisture/100)
  - Ash correction: Y_db = Y_daf * (1 - Ash/100)
  - Original basis preserved in yield_basis column

Conversion coverage:
  - Already dry basis: 76%
  - Converted from daf: 18%
  - Converted from ar: 4%
  - Ambiguous: 2% (assumed dry)

========================================
9. STATISTICAL CONSIDERATIONS
========================================

9.1 Data Distribution
----------------------
Most continuous variables show right-skewed distributions:
  - Yields: Approximately normal with slight right skew
  - Temperature: Bimodal (HTC peak ~220°C, HTL peak ~320°C)
  - Time: Right-skewed (long tail for extended reactions)
  - Compositional data: Bounded [0,100], often left-skewed

Recommendations:
  - Use median for central tendency with skewed data
  - Apply log transformation for regression analysis if appropriate
  - Consider non-parametric methods for hypothesis testing

9.2 Correlation Structure
--------------------------
Strong correlations (|r| > 0.7):
  - C content ↔ HHV (r = 0.89)
  - O/C ↔ H/C (r = -0.72)
  - Temperature ↔ Bio-oil yield (r = 0.58)
  - Lignin ↔ LRI (r = 0.81)

Weak correlations (|r| < 0.3):
  - Residence time ↔ Yields (process-dependent)
  - Pressure ↔ Products (often autogenous, covariate with T)

9.3 Missing Data Patterns
--------------------------
Missingness mechanism: Primarily MNAR (Missing Not At Random)
  - Product composition more likely missing in older studies
  - Gas yields often not reported (measurement difficulty)
  - Catalyst details missing in non-catalytic studies (by design)

Handling strategy:
  - Complete case analysis: Use for sensitivity analysis
  - Imputation: Use for predictive modeling (as documented)
  - Missing indicator: Include *_imputed flags as features

9.4 Sample Size Recommendations
--------------------------------
For machine learning model development:
  - Minimum training set: 500-800 samples per process type
  - Cross-validation: 5-10 fold recommended
  - Test set: 15-20% holdout
  - Stratification: By feedstock family and process type

For statistical analysis:
  - Subgroup analysis: Minimum 30 samples per group
  - Regression: Minimum 10-15 samples per predictor
  - Consider hierarchical models for multi-level structure
